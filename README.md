# GharPay CRM Project

This is a comprehensive CRM project consisting of a React-based frontend and a Node.js/Express backend.

## Structure

- `/pg-lead-hub-main`: The frontend web application built with React, Vite, and TailwindCSS.
- `/server`: The backend REST API built with Node.js, Express, and MongoDB.
- `/screenshots`: Contains screenshots of the application.

## Prerequisites

- Node.js
- MongoDB
